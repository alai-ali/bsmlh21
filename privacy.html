Данные хранятся в Firebase (Google Cloud) с сертификатами безопасности ISO 27001. После перехода на блокчейн — HUID и токен-баланс хранятся децентрализованно и не могут быть изменены или удалены.

  • Запросить информацию о хранимых данных
  • Потребовать исправления неточных данных
  • Удалить аккаунт и персональные данные
  • Отозвать согласие на обработку

Важно: данные в блокчейне (HUID, токены) не могут быть удалены в силу технической природы блокчейна.

Мы передаём данные только в трёх случаях:

  • По законному требованию государственных органов
  • Firebase/Google — для технического хранения (политика Google)
  • В обезличенном виде — для улучшения алгоритмов

Платформа использует localStorage браузера для хранения данных сессии на вашем устройстве (HUID, имя). Эти данные не передаются на сторонние серверы.
